# Rice-Leaf-Disease-Detection
Rice Leaf Disease Detection is a machine learning-based project developed to automatically identify and classify various diseases affecting rice plants by analyzing leaf images. The system uses a trained ML model to detect diseases such as bacterial Leaf blight, brown spot, healthy , hispa , leaf blast and leaf smut with high accuracy.

The project is implemented through two user interfaces:

🌐 A Web Application (developed using Flask) for browser-based predictions

📱 An Android Mobile App for real-time field detection using smartphones

This solution aims to assist farmers and agricultural experts in early disease identification to reduce crop loss and improve productivity.
